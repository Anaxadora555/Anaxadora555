# Sales_Dashboard
__Интерактивный дашборд для отслеживания ключевых бизнес‑метрик компании, продающей офисное оборудование. Объединяет финансовые, операционные и маркетинговые показатели для всестороннего анализа продаж и эффективности бизнеса.__

__Этапы проекта__
  * Определение требований и целей проекта.
  * Сбор данных из источников и их интеграция.
  * Очистка и подготовка данных для анализа.
  * Агрегация и построение моделей данных.
  * Создание расчетных показателей и визуализаций.
  * Настройка интерактивных элементов и фильтров.
  * Тестирование и финальная проверка перед запуском.
