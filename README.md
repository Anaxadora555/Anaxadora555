# Sales_Dashboard
Интерактивный дашборд для отслеживания ключевых бизнес‑метрик компании, продающей офисное оборудование. Объединяет финансовые, операционные и маркетинговые показатели для всестороннего анализа продаж и эффективности бизнеса.

Этапы проекта
1. Определение требований и целей проекта.
2. Сбор данных из источников и их интеграция.
3. Очистка и подготовка данных для анализа.
4. Агрегация и построение моделей данных.
5. Создание расчетных показателей и визуализаций.
6. Настройка интерактивных элементов и фильтров.
7. Тестирование и финальная проверка перед запуском.
